About the Playground

                

                

                    
The Python Playground is a web service inspired to the Go Playground that runs on Google App Engine. The service receives a Python program, runs it inside a sandbox and then returns the output.

                    
To find the Python version used by the the playground, run this program.

                    
The source code is available on Github.
